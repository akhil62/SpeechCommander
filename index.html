SpeechCommander

A Small PyGTK App Implementing CMU Sphinx

Description

This app is used to recognize speech based system commands. A small corpus is included in the package (corpus.txt,commandcorpus.cr). You may edit these by using the settings window

Installing

As of now, this can be run by using the main.py file

  $ python main.py

Using

The main.py file shows two main buttons, Launch App and Launch Settings. On clicking the Launch App button, the App Window is invoked. The speak button is used to give speech commands

Settings

The Launch Settings button invokes the settings window. You may add/edit words in th corpus there.
